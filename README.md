# Karate score board

With this, you no longer need to buy KARATE score board.

## Tested Environment
- macOS
- python 3.7.2

![gif](images/board.gif "gif")

## Usage

1. git clone

`git clone https://github.com/rika77/KARATE_score_board.git`

2. Run

`python3 main.py`
